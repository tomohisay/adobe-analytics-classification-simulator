# Adobe Analytics 分類シミュレーター

Adobe Analyticsの分類（Classification）機能を体験できる教育用Webアプリケーションです。

## 概要

Adobe Analyticsでは、eVarやprop変数で計測した値（キー）に対して、追加の属性情報をマッピングする「分類」機能があります。このシミュレーターでは、ダミーデータを使って分類の概念を直感的に理解できます。

## デモ

ブラウザで `index.html` を開くだけで動作します。

## ユースケース

### 1. 広告トラッキングコード分類
- **キー**: `sem_google_brand_2024` などのトラッキングコード
- **分類属性**: キャンペーン名、媒体、クリエイティブ

### 2. URL分類
- **キー**: `/products/shoes/nike-air-max` などのページURL
- **分類属性**: ページカテゴリ、コンテンツタイプ、製品ライン

### 3. 会員ID分類
- **キー**: `M001234` などの会員ID
- **分類属性**: 会員ランク、登録地域、性別

## 機能

- 元データ（Raw Data）の表示・追加
- 分類テーブルの作成・編集・削除
- 分類結果のリアルタイム生成
- **分析例（Workspace風レポート）**: 分類カラムごとの集計表示
- LocalStorageへのデータ保存
- JSONエクスポート

## 使い方

1. ブラウザで `index.html` を開く
2. 3つのタブ（広告/URL/会員ID）を切り替えて各ユースケースを確認
3. 「分類ルール追加」で新しい分類を登録
4. 「分析例」セクションで集計結果を確認
5. 分類ルールを削除すると、該当データが「未分類」になることを確認

## 技術仕様

- HTML/CSS/JavaScript（フレームワーク不使用）
- 単一ファイルで完結
- データ永続化: LocalStorage
